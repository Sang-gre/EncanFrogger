FOR CMD:
    Quickest way to compile and run:
    run.cmd

    To create a list of all files:
    dir /s /b src\*.java > sources.txt

    To compile contents of said list: 
    javac -d bin @sources.txt

    Run game:
    java -cp bin main.GameLauncher

FOR POWERSHELL:
    Quickest way to compile and run:
    ./run.cmd

    To compile:
    javac -d bin -sourcepath src (Get-ChildItem -Recurse src/*.java)

    To run:
    java -cp bin main.GameLauncher



Don't mind:
javac -d bin -sourcepath src src/main/GameLauncher.java
